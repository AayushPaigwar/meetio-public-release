# Meetio Check-In

- A simple check-in app for event organizers and on-ground teams.
- Helps manage attendee entry during live events.
- Scan attendee QR codes for quick check-ins.
- Search and check in guests manually when needed.
- Track event attendance and recent check-in activity.
- Supports smooth entry management for meetups, conferences, and community events.
- Built as a companion app for organizers using Meetio at meetio.online.
